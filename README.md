# http
Shared String helper functions


#### Installation

1. Add this to composer.json:
```
  "repositories": [
    {
      "type": "vcs",
      "url": "https://github.com/nick-shared/php-string.git"
    }
  }
```
2. Add this to composer.json:
```
"require": {
    "mutant/string": "dev-master"
}
```

3. `composer install`
